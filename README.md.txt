# 🎓 Star Schema para Análise de Professores com Power BI

## 📌 Sobre o Projeto

Este projeto foi desenvolvido como parte do desafio da DIO, cujo objetivo é transformar um modelo relacional em um modelo dimensional utilizando a arquitetura **Star Schema**.

O foco da análise está nos professores, permitindo consultas analíticas mais rápidas e eficientes através de um Data Warehouse.

---

# 🏛 Arquitetura

Modelo Dimensional (Star Schema)

Tabela Fato

- F_Professor

Tabelas Dimensão

- D_Professor
- D_Departamento
- D_Curso
- D_Disciplina
- D_Data
